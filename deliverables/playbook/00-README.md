## How to use this in the next 48 hours

1. Read files 01 → 07 in order once. Takes ~30 minutes.
2. Print file 01 (meeting checklist) to A4 — you'll tick as you go.
3. Skim file 02 before the meeting; keep open on phone during it.
4. Read files 04 and 05 twice — pricing / negotiation is the sale.
5. Use file 07's stack as your mental model during the meeting so you can answer "how does it work?" technically.
